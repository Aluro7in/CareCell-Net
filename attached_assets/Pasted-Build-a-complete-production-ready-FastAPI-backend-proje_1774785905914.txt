Build a complete production-ready FastAPI backend project called "CareCell Network".

PROJECT GOAL:
An AI-powered emergency healthcare system that connects cancer patients with blood and platelet donors and nearby hospitals in real-time.

TECH STACK:
- FastAPI (Python backend)
- PostgreSQL (via SQLAlchemy ORM)
- Pydantic for validation
- Uvicorn server
- Optional: OpenAI API for AI assistant
- Use environment variables for all API keys

PROJECT STRUCTURE:

DATABASE MODELS:
1. Donor:
- id, name, blood_group, latitude, longitude
- last_donation_date, available, phone

2. Patient:
- id, name, blood_group, cancer_type
- latitude, longitude, urgency (normal/critical)

3. Request:
- id, patient_id, status, created_at

4. Hospital:
- id, name, location, beds_available, blood_bank_available

CORE FEATURES:

1. DONOR REGISTRATION API:
POST /donors
- Register donor with validation

2. PATIENT REQUEST API:
POST /requests
- Create emergency request

3. MATCHING ENGINE (IMPORTANT):
- Filter donors by blood group compatibility
- Calculate distance using Haversine formula
- Exclude recent donors (within 90 days)
- Score system:
   blood match = 50
   distance < 5km = 30
   available = 20
- Return top 5 donors sorted by score

4. EMERGENCY ALERT SYSTEM:
- When request is created:
   → automatically match donors
   → send alert (mock SMS function)
- Create notification service (print or log alerts)

5. HOSPITAL FINDER:
GET /hospitals/nearby
- Return nearby hospitals based on location

6. AI ASSISTANT (OPTIONAL):
POST /ai/chat
- Use OpenAI API
- Answer queries like:
   “Find donors near me”
   “What to do in emergency”

Build a full-stack production-ready web app called "CareCell Network".

GOAL:
An AI-powered emergency healthcare platform where cancer patients can request blood/platelet donors and nearby donors receive real-time alerts.

TECH STACK:
- Frontend: React + Tailwind CSS
- Backend: FastAPI (Python)
- Database: SQLite (for simplicity, but structured for PostgreSQL)
- API communication: REST
- Optional AI: OpenAI API (via environment variable)

BACKEND FEATURES:

1. Donor APIs:
- POST /donors → register donor
- GET /donors → list donors

2. Patient Request:
- POST /requests → create emergency request

3. Matching Engine:
- Match donors by:
   blood group
   distance (Haversine)
   availability
- Score:
   blood = 50
   distance <5km = 30
   available = 20
- Return top 5 donors

4. Alert System:
- When request created:
   → match donors
   → send alert (mock print/log)

5. Hospital API:
- GET /hospitals → list hospitals

6. AI Assistant:
- POST /ai/chat
- Uses OpenAI API (env variable)
